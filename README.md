# Cafe_Monitoring_System<br>
<p><b>카메라, 공기질, 소음측정 센서를 이용한 카페 추천 시스템</b></p>
<p>공부, 대화, 휴식 등 각각의 이유로 카페를 찾는 이용자들을 위해 카메라, 공기질, 소음측정 센서를 통해 카페 내부의 환경을 실시간으로 모니터링하여 앱을 통해 사용자에게 알려줌으로써 자신에게 적합한 카페를 선택할 수 있도록 도와준다.</p>
